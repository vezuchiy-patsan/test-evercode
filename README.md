# Тестовое задание Evercode

REST API

## Запуск

1. Клонируйте проект
2. Запустить установку зависимостей

```bash
   npm install
```

3. Запустить проект

```bash
   npm run dev
```

или

```bash
npm start
```

4. Для проверки можете использовать любую программу для тестирования API (Postman)

P.S. данные сохраняются в json файле data.json в корне проекта
